## Project 1: NLP Multilingual Machine Translation System

| Attribute | Details |
|-----------|---------|
| **Domain** | Natural Language Processing / Machine Learning |
| **Duration** | 4 Months |
| **Stipend** | Rs. 10,000 - 15,000 |
| **Positions** | 10-12 Students |

### Project Description

Develop an AI-powered translation system to convert automotive technical documentation from English to Indian languages (Hindi, Tamil, Telugu, Malayalam). Work with state-of-the-art NLP models and cloud infrastructure.

### Required Skills

| Role | Positions | Skills Required |
|------|-----------|-----------------|
| **NLP/ML Engineer** | 4-5 | Python, PyTorch/TensorFlow, Basic ML concepts |
| **Backend Developer** | 2 | Python, REST APIs, SQL databases |
| **Frontend Developer** | 1-2 | React/Vue, JavaScript, HTML/CSS |
| **MLOps/DevOps** | 1-2 | Linux, Docker, Cloud basics (GCP preferred) |
| **QA/Testing** | 1 | Testing fundamentals, Documentation |

### Preferred Qualifications

- Interest in NLP/AI/Machine Learning
- Basic understanding of deep learning concepts
- Familiarity with Python programming
- Self-motivated and quick learner
- Good communication skills

## Project 2: 3D Wiring Power Flow Automation Tool

| Attribute | Details |
|-----------|---------|
| **Domain** | 3D Graphics / CAD / Web Development |
| **Duration** | 2 Months |
| **Stipend** | Rs. 10,000 - 15,000 |
| **Positions** | 8-10 Students |

### Project Description

Build an interactive 3D visualization tool for automotive wiring diagnostics. Create animations showing power flow through truck wiring systems, helping technicians diagnose faults visually.

### Required Skills

| Role | Positions | Skills Required |
|------|-----------|-----------------|
| **3D Developer** | 3-4 | Blender/Maya basics, Unity, 3D modeling concepts |
| **Backend Developer** | 2 | Python, REST APIs, SQL databases |
| **Frontend Developer** | 2 | React, Three.js/WebGL, JavaScript |
| **DevOps** | 1 | Linux, Docker, Cloud basics |
| **QA/Testing** | 1-2 | Testing fundamentals, Documentation |

### Preferred Qualifications

- Interest in 3D graphics, game development, or visualization
- Basic understanding of web technologies
- Familiarity with any 3D software (Blender, Unity, etc.)
- Self-motivated and quick learner
- Good communication skills
